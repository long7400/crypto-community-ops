# Tasks

Store lightweight task briefs for `crypto-community-ops` here.

Suggested filename format:

`YYYY-MM-DD-short-task-name.md`

Use a task brief when:

- the work is small or medium sized
- the scope is already clear
- a full execution plan would be unnecessary overhead

Escalate from a task brief to an execution plan if the task grows, crosses boundaries, or reveals unknowns.

For this repo, prefer a brief before changes involving Telegram onboarding,
Discord runtime events, bot tokens, project-manager team coordination, or social
publishing behavior.
